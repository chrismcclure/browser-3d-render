# browser-3d-render

A browser-based 3D world engine built with React, TypeScript, Vite, Three.js, React Three Fiber, and Drei.

The long-term goal is reusable, data-driven 3D environments. The first environment is a walkable version of a real house.

## Development

```bash
npm install
npm run dev
npm run build
npm run lint
```
